# KSpellacy.info multi-page site

Upload these files to the root of the GitHub Pages repository for kspellacy.info.

Files included:
- index.html
- web-design.html
- portfolio.html
- stars.html
- contact.html
- assets/style.css

Optional image:
- Add `keaton-headshot.png` to the same folder as `index.html` if you want the headshot to appear. If the image is missing, the site shows a KS placeholder.
